# Quick Memo Pad

全画面アプリの上にフロートする軽量メモアプリ（macOS）

## ダウンロード

[Releases](../../releases) から最新の `.dmg` をダウンロードしてください。

### インストール手順（macOS）

署名なしアプリのため、初回は以下の手順が必要です:

1. ダウンロードした `.dmg` を開き、アプリを「アプリケーション」フォルダにドラッグ
2. **ターミナルを開いて以下のコマンドを実行:**
   ```
   xattr -cr /Applications/Quick\ Memo\ Pad.app
   ```
3. アプリケーションフォルダから Quick Memo Pad を開く
4. 以降は普通にダブルクリックで起動できます

> **「"Quick Memo Pad"は壊れているため開けません」と表示された場合**
> 上記の `xattr -cr` コマンドを実行してから再度開いてください。

## 特徴

- **全画面オーバーレイ** — フルスクリーンアプリの上にメモを表示可能
- **透過度調整** — ウィンドウを半透明にして背景を見ながらメモ
- **ホットキー** — `⌘+Shift+A` でどこからでも瞬時に呼び出し
- **自動保存** — 入力内容は自動的にローカルに保存
- **メモ検索** — 書き溜めたメモをリアルタイム検索
- **テーマ4種** — クラシック / ダーク / ブルー / ウォーム
- **フォント変更** — 6種類のフォント、7段階のサイズ
- **トレイ常駐** — ウィンドウを閉じてもメニューバーに常駐

## キーボードショートカット

| ショートカット | 動作 |
|---------------|------|
| `⌘+Shift+A` | 表示/非表示（グローバル） |
| `⌘+Shift+P` | 全画面の上に固定 ON/OFF |
| `⌘+N` | 新規メモ |
| `⌘+L` | メモ一覧 |
| `⌘+G` | Web検索 |
| `⌘+,` | 環境設定 |
| `⌘+[` / `⌘+]` | 透過度 調整 |
| `Escape` | パネルを閉じる |

## 開発

```bash
npm install

```
